# hulu-clone-enhanced
A modern, responsive Hulu-inspired streaming platform clone built for learning and showcasing front-end and full-stack development skills. This project enhances the original Hulu clone with new features, improved UI, and better performance.

## Features
- Responsive Design – Works seamlessly on desktop, tablet, and mobile devices.
- Enhanced UI/UX – Improved layout, animations, and hover effects for a modern streaming experience.
- Movie & Series Listings – Displays trending, popular, and recommended content.

## Tech Stack
- HTML
- CSS
- JAVASCRIPT
